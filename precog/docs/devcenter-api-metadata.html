REST API

metadata api

The metadata API provides a one-stop resource for metadata on the structure of data, statistics on queries, and so forth.

Retrieve Path Metadata

JSON
GET
GET /meta/v1/fs/'path?type=[type]&apiKey=[auth API key]
description

Retrieves metadata for a particular path in the file system.

The valid metadata types include size, structure, and children. If no metadata type is specified, then all metadata is returned.
